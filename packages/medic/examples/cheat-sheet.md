# Medic Cheat Sheet

## Posted via [@minodisk/medic](https://github.com/minodisk/medic)

### Block Elements

#### Paragraphs and Line Breaks

In markdown:

```
Foo
Bar

Baz
```

In Medium:

Foo
Bar

Baz

#### Headers

In markdown:

```
### Foo
#### Bar
```

In Medium:

### Foo
#### Bar

#### Horizontal Rules

```
Foo

---

Bar
```

Foo

---

Bar

#### Blockquotes

In markdown:

```
> Your text
```

In Medium:

> Your text

In markdown:

```

2. Bar
3. Baz

- Bar
- Baz

```
